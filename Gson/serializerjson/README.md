# serializejson

Projeto Java com Maven para demonstrar serialização de objeto em JSON utilizando a biblioteca Gson.

## Tecnologias
- Java 21+
- Maven
- Gson 2.13.2
- VSCode

## Estrutura
- `Pessoa`: classe com `name` e `idade`
- `Main`: cria um objeto `Pessoa` e serializa para JSON com Gson

## Exemplo de saída
```json
{"name":"Rodrigo","idade":20}
